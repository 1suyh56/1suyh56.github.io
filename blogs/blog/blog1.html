# 1.博客的搭建

第一次尝试搭建自己的博客，所以第一篇就把搭建的经验放在这里，作为博客的开头吧，另外，我也没想着有人到访，留言板就不做了(～￣▽￣)～(懒.jpg)
由于github经常推不上去，所以放弃了github+hexo的选型，改了gitee和vuepress，正赶上绿网，所以暂时推迟和大家见面了，希望等人发现的时候它已经搭建完成了
图片使用图床的形式发布，选型GitHub和PicGo同样分享给大家，看下一篇文章即可
轮子是在前人的基础上进行改造，原地址在这里
我的在这里,开箱，加载yarn，就可以使用了

# 2.配置讲解

/.vuepress/config文件中
title和description对应加载页标题和描述


head中link可以指定标题的小图标

plugins对应插件，可以从vuepress插件市场引用

这里是信息栏，可以对个人图片之类的设置进行设置，logo和author，对应左上角的小图和姓名，改成自己即可，authorAvatar对应右侧边大图，其余都是底部栏的设置

# 3.路由设置

/.vuepress/siderbar这个配置比较简单，一目了然，配合目录和博客右侧边栏看看就懂了

const { createSideBarConfig } = require('./util')
const XIAOCHENGXU_PATH = '/blogs/xiaochengxu'
const HELLO_PATH = '/blogs/blog'
module.exports = {
  [XIAOCHENGXU_PATH]: [createSideBarConfig('小程序', XIAOCHENGXU_PATH)],
  [HELLO_PATH]: [createSideBarConfig('博客', HELLO_PATH)],
}
